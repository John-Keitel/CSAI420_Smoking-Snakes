# React UI - On-Demand Skills

When building, modifying, or reviewing UX/UI (React components, pages, forms, styles, layouts):

1. Read `.agents/skills/react-best-practices/SKILL.md` - performance, waterfalls, bundle size, re-renders.
2. Read `.agents/skills/web-design-guidelines/SKILL.md` - accessibility, focus, forms, semantics, copy.
3. For component architecture (compound components, boolean-prop refactors), also read `.agents/skills/react-composition-patterns/SKILL.md`.

Load compiled rule details only when needed: `references/full-guide.md` inside each skill, or individual files under `rules/`.

Do not assume always-on access to the full compiled guides - they are intentionally on-demand to preserve context.
